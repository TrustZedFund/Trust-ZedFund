How Trust ZedFund Works

Users define rules. Trust ZedFund enforces them. Funds move only when conditions are met.

Start Alone — It’s Normal

  • Create an account
  • Create a fund you control initially
  • Name it by purpose
  • Set safety rules early
  • Invite others later

Trust ZedFund enforces user-defined rules and does not make financial decisions.
